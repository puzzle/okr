This is the API

createObjective

Create a new Objective

URL

http://localhost:8080/api/v2/objectives

HTTP Method

POST

Response Type

ObjectiveDto

Parameters

  • Body: ObjectiveDto ObjectiveDto(ObjectiveDto)

    The Objective as json to create a new Objective.

deleteObjectiveById

Delete Objective by ID

URL

http://localhost:8080/api/v2/objectives/{id}

HTTP Method

DELETE

Response Type

Parameters

  • Path: id Long

    The ID of an Objective to delete it.

duplicateObjective

Duplicate a given Objective

URL

http://localhost:8080/api/v2/objectives/duplicate

HTTP Method

POST

Response Type

ObjectiveDto

Parameters

  • Body: DuplicateObjectiveDto DuplicateObjectiveDto(DuplicateObjectiveDto)

    The Objective which should be duplicated as JSON

getKeyResultsFromObjective

Get all KeyResults from one Objective by ObjectiveId.

URL

http://localhost:8080/api/v2/objectives/{id}/keyResults

HTTP Method

GET

Response Type

Object

Parameters

  • Path: id Long

    The ID for getting all KeyResults of an Objective

getObjective

Get an Objective by ID

URL

http://localhost:8080/api/v2/objectives/{id}

HTTP Method

GET

Response Type

ObjectiveDto

Parameters

  • Path: id Long

    The ID for getting an Objective.

updateObjective

Update Objective by ID

URL

http://localhost:8080/api/v2/objectives/{id}

HTTP Method

PUT

Response Type

ObjectiveDto

Parameters

  • Path: id Long

    The ID for updating an Objective.

  • Body: ObjectiveDto ObjectiveDto(ObjectiveDto)

    The objective as json to update an existing Objective.
