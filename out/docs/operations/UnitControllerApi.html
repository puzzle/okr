This is the API

createUnit

Creates a new unit with the provided details.

URL

http://localhost:8080/api/v2/units

HTTP Method

POST

Response Type

UnitDto

Parameters

  • Body: UnitDto UnitDto(UnitDto)

    Unit data in JSON format.

deleteUnitById

Deletes an existing unit by its ID.

URL

http://localhost:8080/api/v2/units/{unitId}

HTTP Method

DELETE

Response Type

Parameters

  • Path: unitId Long

    ID of the unit to be deleted.

getAllUnits

Retrieves a list of all available units.

URL

http://localhost:8080/api/v2/units

HTTP Method

GET

Response Type

UnitDto

Parameters

getUnitsByUser

Retrieves a list of units associated with the currently authenticated user.

URL

http://localhost:8080/api/v2/units/user

HTTP Method

GET

Response Type

UnitDto

Parameters

updateUnit

Updates an existing unit with new details.

URL

http://localhost:8080/api/v2/units/{unitId}

HTTP Method

PUT

Response Type

UnitDto

Parameters

  • Path: unitId Long

    ID of the unit to be updated.

  • Body: UnitDto UnitDto(UnitDto)

    Updated unit data in JSON format.
