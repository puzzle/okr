This is the API

addUsersToTeam

Add users to a team

URL

http://localhost:8080/api/v2/teams/{id}/addusers

HTTP Method

PUT

Response Type

Parameters

  • Path: id Long

    The ID of an Team to add to users to it.

  • Body: UserDto List(UserDto)

createTeam

Create a new Team

URL

http://localhost:8080/api/v2/teams

HTTP Method

POST

Response Type

TeamDto

Parameters

  • Body: TeamDto TeamDto(TeamDto)

    The Team as json to create a new Team.

deleteTeamById

Delete Team by ID

URL

http://localhost:8080/api/v2/teams/{id}

HTTP Method

DELETE

Response Type

Parameters

  • Path: id Long

    The ID of an Team to delete it.

getAllTeams

Get all Teams from db

URL

http://localhost:8080/api/v2/teams

HTTP Method

GET

Response Type

TeamDto

Parameters

removeUserFromTeam

Remove User with given UserID from Team

URL

http://localhost:8080/api/v2/teams/{id}/user/{userId}/removeuser

HTTP Method

PUT

Response Type

Parameters

  • Path: id Long

    The ID of an team to remove the user from it.

  • Path: userId Long

    The User ID to remove from the team.

updateOrAddTeamMembership

If user is already member of this team, isAdmin is set. otherwise new team membership is added with isAdmin true or false

URL

http://localhost:8080/api/v2/teams/{id}/user/{userId}/updateaddteammembership/{isAdmin}

HTTP Method

PUT

Response Type

Parameters

  • Path: id Long

    The ID of an team to update or add membership

  • Path: userId Long

    The User ID to update or add membership

  • Path: isAdmin Boolean

    The parameter if user should be admin or not
