This is the API

createCheckIn

Create a new Check-in

URL

http://localhost:8080/api/v2/checkins

HTTP Method

POST

Response Type

Object

Parameters

  • Body: body Object(object)

    The Check-in as json to create a new Check-in.

deleteCheckInById

Delete Check-in by ID

URL

http://localhost:8080/api/v2/checkins/{id}

HTTP Method

DELETE

Response Type

Parameters

  • Path: id Long

    The ID of an Check-in to delete it.

getCheckInById

Get Check-in by ID

URL

http://localhost:8080/api/v2/checkins/{id}

HTTP Method

GET

Response Type

Object

Parameters

  • Path: id Long

updateCheckIn

Update a Check-in by ID

URL

http://localhost:8080/api/v2/checkins/{id}

HTTP Method

PUT

Response Type

Object

Parameters

  • Path: id Long

    The ID for updating a Check-in.

  • Body: body Object(object)

    The Check-in as json to update an existing Check-in.
